History of CSS

Versions of CSS

CSS1

CSS1: the very first version of CSS and Recommendation of W3C. it was launched in 1996 with capablities of font properties. it also used for adding background color and text side

CSS2

W3C developed the next version of CSS and named it as CSS2. it was launched in 1998. it has more feature and functionalities than the previous version.

CSS3

CSS3 is the largest version of CSS officially by W3C, and it was launched in 1999. It has vast collection of font type and you can use any font type from Google and typecast. Currently, most companies and organization us CSS3 and HTML5 for web development and designing task.

Types of CSS(Cascading Style Sheet)

There are three types of CSS which are given below:
  • Embedded CSS
  • Inline CSS
  • External CSSS
  • Import CSS
Embedded CSS:This can be used when a single HTML document must be styled uniquely. The CSS rule set should be within the HTML file in the head section i.e the CSS is embedded within the HTML file.
Example: Embedded CSS

Inline CSS:Inline CSS contains the CSS property in the body section attached with element is known as inline CSS. This kind of style is specified within an HTML tag using the style attribute.
Example: Inline CSS
